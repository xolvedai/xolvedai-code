# Coding Intelligence x XolvedAI

**The ultimate xAI Grok-powered coding agent for full-stack development.**  
Native tool calling, multi-file batch edits, vision support, approval workflows, and Cline-style autonomous workflows — all powered by Grok 4.20-reasoning.

Better than Claude Code because it's built for **xAI** from the ground up.

## Features
- 🚀 **Grok 4.20-reasoning** with native tool calling
- 📁 Multi-file batch editing with approval
- 👁️ Vision / screenshot support (coming in next update)
- ✅ Plan / Act mode toggle
- 🔑 Works with `XAI_API_KEY` (env var or `.env.local`)
- ⚡ One shortcut: `Ctrl+Shift+G` (or `Cmd+Shift+G` on Mac)

## Installation
1. Install from the [VS Code Marketplace](https://marketplace.visualstudio.com/items?itemName=XolvedAI.coding-intelligence-x-xolvedai)
2. Set your xAI API key (see below)
3. Press `Ctrl+Shift+G` to open the agent chat
